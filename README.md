# charlie-shell
A custom Python-based command shell with built-in file operations, 
command history, and a modular architecture. Designed to evolve from a CLI into a GUI-based terminal for real-world system-level usage.

Day 1: 
Learned about basic features of OS module of python.
Gather information about the commands and its parameters.
Made a basic understanding of working of shell and its communication with kernal.
used the basic commands like os.mkdir(), os.remove(), os.getcwd(), etc.

Day 2: 
Make a basic working shell with open, exit, make folder, delete folder, go inside a folder (cd).
Spend a day on that stuff with applying different logic and custom wording for the commands
